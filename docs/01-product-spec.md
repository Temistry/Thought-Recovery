# 제품명(가칭): Idea Second Brain

## 한 줄 정의
말하거나 짧게 적으면, AI가 아이디어를 버려지지 않게 정리·연결·구체화하는 음성 중심 세컨드브레인 앱.

## 핵심 문제
일반 사용자는 메모를 남기지만 시간이 지나면 뒤로 밀려 잊어버린다. Notion/Obsidian처럼 강력한 도구는 직접 정리해야 해서 유지보수가 어렵다.

## 핵심 해결
사용자는 입력만 한다. 앱은 원본을 보존하고, AI가 제목/요약/태그/관련 메모/구체화 질문/위키 링크를 자동 제안한다.

## 타깃 사용자
- 아이디어는 많지만 정리를 못 하는 일반 사용자
- 게임/콘텐츠/유튜브/창업 아이디어를 자주 떠올리는 사람
- Notion/Obsidian/메모앱을 쓰다 실패한 사람
- 음성으로 빠르게 생각을 남기고 싶은 사람

## MVP 목표
1. PC/모바일에서 같은 계정으로 메모 동기화
2. 텍스트 메모 입력
3. 음성 메모 입력 시작점 제공
4. 원본 텍스트 보존
5. AI 정리 필드가 들어갈 데이터 구조 확보
6. 관련 메모 연결 구조 확보

## 제품 원칙
- 원본은 절대 덮어쓰지 않는다.
- AI 편집은 항상 제안/파생 데이터로 저장한다.
- 사용자는 복잡한 편집 UI 없이 입력, 확인, 거절, 더 구체화만 한다.
- 그래프 뷰는 후순위다. 먼저 “다시 찾을 수 있음”과 “자동 연결”을 만든다.

## 첫 번째 성공 기준
아이폰에서 메모를 남기고 PC 웹에서 바로 같은 메모가 보인다.
